Read-only IAM credentials

🔐 We only need read access. Create a read-only IAM user with ReadOnlyAccess policy. Takes 2 minutes.   Open AWS IAM Console →

Google Cloud Service Account

🔐 Create a service account with Viewer role + Cloud Asset Viewer. Download the JSON key file.   Open GCP Console →
Paste the contents of your service account .json key file

Azure Service Principal

🔐 Register an app in Azure AD, assign the Reader role at subscription scope, and create a client secret.   Open Azure Portal →

Snowflake Account

❄️ Connect with an ACCOUNTADMIN or SYSADMIN role for full cost visibility. We only run read-only queries.   Open Snowflake →

Leave blank to use ACCOUNTADMIN. Needs access to SNOWFLAKE.ACCOUNT_USAGE schema.

Cloudflare API Token

Oracle Cloud (OCI) API Key
